## 目录说明

### src/app/api/
- 存放Next.js API路由，对外暴露的接口
- 使用App Router的路由系统

### src/lib/
- 存放通用工具库
- 包括数据库连接、Redis连接等基础服务

### src/modules/
- 按业务模块组织代码
- 每个模块包含API、服务、类型等

### src/modules/music/
- 音乐业务模块
- 包含对外API接口和内部服务

### src/modules/music/services/providers/
- 对接不同音乐平台的实现
- 每个平台一个目录，实现统一的接口

### doc/
- 存放项目文档
- 包括原始Java代码、API文档和目录结构说明

### prisma/
- Prisma ORM配置
- 定义数据库模型

## 设计模式说明

1. **策略模式**：
   - 定义统一的音乐服务接口
   - 不同服务商实现各自的策略
   - 便于快速切换和扩展服务商

2. **工厂模式**：
   - 根据平台类型创建对应的服务商实例
   - 封装实例创建逻辑

3. **适配器模式**：
   - 适配不同服务商的API返回格式
   - 统一对外输出格式

4. **单例模式**：
   - 数据库连接池
   - Redis连接
   - 确保资源的高效使用

## 技术栈

- **前端框架**：Next.js 16.1.4
- **后端语言**：TypeScript
- **ORM框架**：Prisma
- **网络请求**：Axios
- **数据库**：MySQL
- **缓存**：Redis
- **样式**：Tailwind CSS
- **部署**：Vercel
